## Versioning

Every deployment must include a version number:
- **In the app**: Display in the UI (footer, `/version` endpoint, or about page)
- **In GitHub**: Tag the release (`git tag vX.Y.Z && git push --tags`)
- **In Notion**: Record the version number in the project's Recent Changes section

## Content Model

### Item Types
- **Advice**: (New) Practical guidance, suggestions, or wisdom for action (e.g., business, career, or life decisions).
- **Other types**: Quote, Affirmation, Story, Thought, Lesson, Habit.

### Tagging Conventions
- **Buddhism**: All items related to Buddhist insights (e.g., Ajahn Chah, Ajahn Brahm) MUST be tagged with `buddhism`.
- **Business**: Advice related to work, marketing, or strategy should be tagged with `business`.
